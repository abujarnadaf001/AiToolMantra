# CONTAINERIZATION RULES

Use Docker for:
- Consistent environments
- Reproducible deployments
- Dependency isolation

Docker systems should:
- Use lightweight images
- Use multi-stage builds
- Minimize attack surface

Avoid:
- Bloated containers
- Running as root
- Massive images
